Import Mermaid dans Draw.io (extension VS Code)

1. Ouvre Draw.io dans VS Code.
2. Cree un nouveau diagramme vide.
3. Va dans Arrange > Insert > Advanced > Mermaid.
4. Colle le contenu de use_case_diagram.mmd ou class_diagram.mmd.
5. Valide pour generer le diagramme.
6. Ajuste les styles/couleurs puis sauvegarde en .drawio.

Fichiers Mermaid generes:
- diagrams/use_case_diagram.mmd
- diagrams/class_diagram.mmd
